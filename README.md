Tên đề tài: Ứng dụng cảnh báo có sự thay đổi file/thư mục từ xa

Mô tả các chức năng chính:

- Theo dõi thời gian thực các sự kiện (Tạo mới, Xóa, Đổi tên, Chỉnh sửa nội dung) trên một thư mục được chỉ định ở máy Client.

- Gửi ngay lập tức cảnh báo (kèm thời gian và tên file) từ Client về Server qua môi trường mạng.

- Server hiển thị log (nhật ký) các thay đổi trên giao diện và lưu trữ lại để tra cứu.

- Chức năng chạy ngầm (background) ở máy Client để liên tục giám sát.
